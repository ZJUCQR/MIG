# mvdream_diffusers Linux 服务器一键复制执行版

这份是 **最少步骤** 的 Linux 服务器运行指南。
目标只有一个：**从 0 开始建环境，然后直接把项目跑起来。**

适用前提：
- Linux 服务器
- NVIDIA GPU
- 已安装 Conda
- 能访问 Hugging Face

---

## 用前先改 1 处路径

把下面命令里的：

```bash
/path/to/mvdream_diffusers
```

替换成你服务器上的真实项目路径。

---

## 方案 A：CUDA 12.1 一键执行

直接整段复制到终端执行：

```bash
REPO=/path/to/mvdream_diffusers && \
cd "$REPO" && \
eval "$(conda shell.bash hook)" && \
conda create -n mvdream python=3.10 -y && \
conda activate mvdream && \
python -m pip install --upgrade pip && \
pip install torch torchvision torchaudio --index-url https://download.pytorch.org/whl/cu121 && \
pip install -r requirements.txt && \
pip install ninja && \
pip install xformers && \
python -c "import torch, diffusers, transformers, accelerate, xformers, kiui; print('torch=', torch.__version__); print('cuda=', torch.cuda.is_available()); print('device=', torch.cuda.get_device_name(0) if torch.cuda.is_available() else 'NO CUDA')" && \
python run_mvdream.py "a cute owl"
```

跑完后，当前目录应该会生成：

```text
test_mvdream_0.jpg
test_mvdream_1.jpg
test_mvdream_2.jpg
test_mvdream_3.jpg
test_mvdream_4.jpg
```

## 跑 ImageDream

如果你想跑图片输入版本，不用重新配环境，激活环境后执行：

```bash
REPO=/path/to/mvdream_diffusers && \
cd "$REPO" && \
eval "$(conda shell.bash hook)" && \
conda activate mvdream && \
python run_imagedream.py data/anya_rgba.png --prompt "a toy character"
```

输出文件会是：

```text
test_imagedream_0.jpg
test_imagedream_1.jpg
test_imagedream_2.jpg
test_imagedream_3.jpg
test_imagedream_4.jpg
```

---

## 如果 `pip install xformers` 失败

把上面命令里的这一行：

```bash
pip install xformers
```

改成：

```bash
pip install -v -U git+https://github.com/facebookresearch/xformers.git@main#egg=xformers
```

然后重新执行后半段安装流程。
